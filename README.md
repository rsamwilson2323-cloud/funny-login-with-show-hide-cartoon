# 😄 Funny Login with Show/Hide Cartoon

**An Interactive Cartoon Login Form Built with HTML, CSS & JavaScript**

🎭 Animated cartoon reactions • 🔐 Interactive login form • ✨ Input-based character expressions • ⚡ Fun user experience

Funny Login with Show/Hide Cartoon is a creative frontend project that transforms a traditional login page into an engaging experience. The cartoon character reacts dynamically when users focus on different input fields, making authentication interfaces more enjoyable and interactive.

Designed for learning **JavaScript DOM manipulation, event handling, UI animations, and creative frontend development**.

---

# ✨ Features

🎭 **Interactive Cartoon Character**

The cartoon changes expressions based on user interactions.

📧 **Email Field Animation**

Character reacts when the email input is selected.

🔐 **Password Field Animation**

Different expression appears when entering a password.

⚡ **Real-Time UI Interaction**

Instant visual feedback improves user engagement.

🎨 **Clean Modern Design**

Simple, elegant, and beginner-friendly interface.

📱 **Responsive Layout**

Works across desktop, tablet, and mobile devices.
